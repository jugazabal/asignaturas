## Prompt 1 (Síntesis ejecutiva)
```
Actúa como analista de debates académicos en orientación educativa.

Marca con [?] cualquier dato incierto.
```
Salida inicial (extracto): "El debate evidencia coincidencias claras: (a) centralidad del enfoque competencial..."
Correcciones manuales: Ajuste de longitud, eliminación de marcador redundante, precisión de términos (CPD añadido).

## Prompt 2 (Tabla analítica ampliada)
```
Genera una tabla Markdown con columnas: Dimensión / Eje | Consensos / Fortalezas | Retos / Brechas | Evidencias (autoría abreviada) | Implicación.
Usa notas: competencias, formación inicial, especialización, oferta formativa, ámbitos, reconocimiento profesional, consenso internacional, lifelong guidance, practicum.
Si falta autor poner [sintetizado].
Limita cada celda a ~20 palabras.
```
Salida inicial: Tabla con algunas celdas >20 palabras.
Correcciones manuales: Recorte de celdas >20 palabras, uniformidad de verbos en infinitivo.

## Prompt 3 (Hilos temáticos)
```
Clasifica el contenido del debate en 4 hilos: formación inicial, competencias, especialización y formación continua, profesionalización y ámbitos.
Para cada hilo redacta 5-6 líneas sintéticas sin repetir párrafos literales.
```
Salida inicial: Hilos con 7-8 líneas.
Correcciones manuales: Reducción longitud, ajuste terminología (diversificación vs expansión), eliminación redundancias.

## Prompt 4 (Conclusión integradora)
```
Redacta conclusión integradora con 5 palancas de mejora. Máx 140 palabras. Español académico claro.
```
Salida inicial: 160 palabras.
Correcciones manuales: Reducción a 139 palabras, sustitución de "capacitación" por "acreditación".
