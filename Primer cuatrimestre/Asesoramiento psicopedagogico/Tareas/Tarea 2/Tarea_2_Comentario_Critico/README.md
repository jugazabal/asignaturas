# Tarea 2: Comentario Crítico - Montanero (2014)

## Proyecto Quarto

Este proyecto contiene el comentario crítico del artículo de Montanero (2014) "La investigación sobre el asesoramiento educativo en España: una revisión de su metodología y resultados empíricos".

### Archivos principales

- **tarea2.qmd**: Archivo fuente del comentario crítico
- **_quarto.yml**: Configuración del proyecto Quarto
- **references.bib**: Referencias bibliográficas en formato BibTeX
- **apa.csl**: Estilo de citación APA (7.ª edición)

### Estructura del comentario

1. **Resumen analítico**
   - Objetivo principal del artículo
   - Principales hallazgos y conclusiones
   - Tipos de metodologías revisadas

2. **Análisis metodológico**
   - Evaluación de los enfoques metodológicos
   - Discusión sobre validez y fiabilidad
   - Reflexión sobre métodos cualitativos vs. cuantitativos

3. **Valoración crítica**
   - Aportaciones del artículo
   - Limitaciones detectadas
   - Propuestas de mejora y líneas futuras

### Compilación

Para generar el PDF, ejecuta en el terminal:

```bash
quarto render tarea2.qmd
```

El archivo PDF se generará en la carpeta `_output/`.

### Especificaciones técnicas

- **Extensión máxima:** 5 páginas
- **Formato:** PDF (A4, 12pt, márgenes 2.5cm)
- **Interlineado:** 1.5
- **Estilo de citación:** APA 7.ª edición
